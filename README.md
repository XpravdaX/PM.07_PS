# PM.07_PS - Система учёта заказов

## О проекте

Информационная система для автоматизации учёта заказов компании «ТехноСервис Плюс». Разработана в рамках производственной практики. Позволяет вести учёт клиентов, товаров, заказов, отслеживать статусы выполнения заказов и формировать отчётность.

## Функциональные возможности

- Авторизация и регистрация пользователей
- Хэширование паролей (SHA-256 + соль)
- Ролевая модель: администратор, менеджер, кладовщик
- CRUD-операции для заказов, клиентов, товаров
- Просмотр заказов через VIEW
- Смена статуса заказа через хранимую процедуру
- Статистика менеджеров через VIEW
- Аудит изменений статуса через триггер

## Подключение к БД

Настройки в `modules/db_config.py`:

```python
_config = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',
    'password': '',
    'database': 'order_accounting'
}
```
